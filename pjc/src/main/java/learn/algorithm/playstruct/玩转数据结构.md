### 二叉树的遍历

#### 三种遍历方式

java语言实现：https://zhuanlan.zhihu.com/p/25632253

(1). 先序遍历

(2). 中序遍历

(3). 后序遍历

三种遍历方式，也就是遍历的顺序不一样。

先序遍历: "根左右"，遍历的顺序: 根节点->左节点->右节点。

中序遍历: "左根右"，遍历的顺序: 左节点->根节点->右节点。

后序遍历: "左右根"，遍历的顺序: 左节点->右节点->根节点。
